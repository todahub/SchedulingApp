## 概要

本アプリは、友人同士のカジュアルな日程調整を効率化するためのコメントベースの日程調整アプリです。

LINE などでの調整では、返信のずれや後出しの条件によって候補整理に時間がかかります。既存の ○× 形式のツールでは、「夜なら行ける」「たぶん行ける」のような曖昧な表現を扱いにくいという課題があります。

このアプリでは、参加者がコメントで可否や条件を入力し、その内容を段階的に解釈して日程候補を整理することを目指しています。現在は開発途中であり、コメント解釈とランキングの統合は未完了です。

## ページ構成

### イベント作成ページ

- イベント名、期間、時間設定を入力できる
- 作成後は参加ページへ遷移する
- 主催者の役割はイベント作成に限定している

### 参加ページ

- 名前を入力して日程を送信できる
- 同じ名前で送信すると内容を更新できる
- カレンダーで日付を選ぶことでコメント入力を補助する
- コメントで参加可否や条件を入力できる

### 結果ページ

- 参加者の入力結果を確認できる
- 全会一致や候補日を表示する設計で実装を進めている
- 現時点ではコメント解釈とランキングの統合が未完了のため、一貫した日程提案は未完成

## 利用フロー

1. イベントを作成する
2. 参加ページの URL を共有する
3. 参加者が名前とコメントを入力する
4. 結果ページで内容を確認する

## 技術的な工夫

- コアの日時抽出と可否判定は辞書ベースで実装し、再現性とテスト容易性を優先している
- LLM は全文解釈には使わず、ラベル間の関係整理や補助的な解釈に限定している
- 全文を LLM に任せると出力の揺れが大きくなるため、役割を意図的に制限している
- 辞書ベースでは拾いきれない表現は、段階的に辞書拡張と限定的な補完処理で対応する設計にしている
- コメント解釈は回帰テストとガードレールテストを追加しながら段階的に強化している
- ランキングは全会一致を優先し、その後に条件付き候補や妥協候補を扱う階層的な設計を採用している
- UI は主催者の操作をイベント作成に寄せ、参加者はコメント中心でチャット感覚に近い入力ができるようにしている
- カレンダー入力補助を用意しつつ、最終的な入力はコメント中心にすることで参加者の負担を下げている
